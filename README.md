#### 使用说明


- QQ授权登录


    > 偶然发现浏览器在JD登录页有个QQ授权登录，可以拉起手机QQ授权，于是捣鼓了一下，发现如果在【JdWeb】APP中使用QQ授权登录，QQ授权后，会跳转到手机的默认浏览器；
    > 额。。？这样我没办法拿到CK了啊。。 最后妥协了，直接在手机 设置 -> 应用管理 -> 默认应用管理 -> 浏览器  ，把【JdWeb】APP设置成了默认浏览器，再测试了一次，发现成功了。
    > 总的看起来，非常鸡肋。聊胜于无吧，先记录下。
